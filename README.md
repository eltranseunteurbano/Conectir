# Pasos para ejecutar el proyecto
1. Instalar las dependencias y las dependecias de desarrollo con el siguiente comando

    ``` 
    $ npm install 
    ``` 
   
2. Correr el proyecto con el siguiente comando
    ``` 
    $ npm run start 
    ```
    

# Comandos
